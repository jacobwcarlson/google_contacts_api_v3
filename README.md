# GMail for Ruby

A Rubyesque interface to the Google Contacts API (version 3).

## Author(s)

* Jacob Carlson

## Installation

Install it manualy:

    git clone git://github.com/jacobwcarlson/google_contacts_api_v3.git
    cd google_contacts_api_v3.git 
    rake install
